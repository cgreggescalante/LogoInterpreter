LogoInterpreter
===============

A Python implementation of the [Logo](https://el.media.mit.edu/logo-foundation) language.

## Basics

- Functions - to func_name ... end
- Loops - repeat n ... end
- Random - pick_random arr
- Output - print variable
- Mathematics 
## Turtle Graphics

### Movement

- Forward - forward, f
- Backward - backward, b
- Right - right, r
- Left - left, l
